# 渠道级别激活 {#channel-level-activation-single-event-playback}

本页介绍渠道中所用资产的激活级别。

本节将介绍以下主题：

* 概述
* 激活窗口
* 将渠道级激活用作单次事件回放
* 处理渠道中资产的重复
   * 分时段
   * 分周

   * 部分组合
* 将渠道级激活用作单次事件回放

## 概述 {#overview}

***渠道级别激活*** ，允许渠道在特定设置计划后进行切换。 单个事件渠道在设置计划后替换主渠道并播放特定时间，直到主渠道再次播放其内容。

以下示例重点介绍以下关键术语，以提供解决方案：

* 全 ***局序列的主序列渠道*** （英文）
* 一 ***个事件渠道*** ，在设置的时间只运行一次
* 为 ***主序列计划内*** 的单个播放事件设置和优先级渠道

## 处理渠道中资产的重复 {#handling-recurrence-in-assets}

您可以在渠道中计划资产，以按照您的要求，按每天、每周或每月的特定时间间隔重复使用。

假定您希望仅在周五下午1:00至晚上10:00显示渠道的内容。 您可以使用 **激活** 选项卡来设置资产所需的重复时间间隔。

### 分时段 {#day-parting}

1. 选择渠道，然后单 **击操** 作栏中的仪表板以打开渠道仪表板。

1. 在从“开始分配”对话框输入渠道日期／时间和 **结束／日期时间** ，您可以使用表达式或自然文本版本指定重复计划。

   >[!NOTE]
您可以跳过或包含“ **从活动** ”和“ **** 活动到计划”字段，并根据您的要求将表达式添加到“活动”字段。

1. 在表达式中输 **入计划** ，您的资产将按特定的日期和时间间隔显示。

#### 分时段表达式示例 {#example-one}

下表总结了在为显示屏分配表达式时可以添加到计划的几个示例渠道。

| **表达式** | **解释** |
|---|---|
| 早8点之前 | 渠道中的资产每天早8点之前播放 |
| 下午2:00后 | 渠道中的资源每天下午2:00后播放 |
| 12:15后和12:45前 | 渠道中的资源每天下午12:15后播放30分钟 |
| 12时15分之前也是12时45分之后 | 渠道中的资源每天在下午12:15之前播放，然后在下午12:45之后播放 |
| 周一、周二、周三或周一周三 | 资产在周一至周三的渠道中播放 |
| 一月一号晚两点以后，也是一月二号，也是一月三号凌晨三点之前 | 1月1日下午2点以后播放的渠道开始中的资产将在1月2日全天播放，一直持续到1月3日凌晨3点 |
| 1月1-2日下午2:00，也是1月2-3日凌晨3:00 | 1月1日下午2点后，渠道开始播放器中的资产继续播放到1月2日凌晨3点，然后在1月2日下午2点再次开始，并持续播放到1月3日凌晨3点 |

>[!NOTE]
您还可以使 _用Milital_ time（即14:00）代替am/pm ** （即下午2:00）。
